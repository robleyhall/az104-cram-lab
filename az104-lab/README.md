## Quick Start

### 1. Check Prerequisites

```bash
./prerequisites/check-prerequisites.sh
```

### 2. Deploy Foundation

```bash
./scripts/deploy-module.sh 00-foundation
```

### 3. Deploy Modules in Order

Modules have dependencies — deploy in this order:

```
00-foundation          ← Deploy first (shared infrastructure)
├── 01-identity        ← Uses resource groups
├── 02-governance      ← Uses resource groups
├── 03-networking      ← Uses foundation VNet
│   ├── 04-dns-connectivity  ← Uses VNets and subnets
│   │   └── 06-storage       ← Uses private endpoints, service endpoints
│   ├── 05-load-balancing    ← Uses VNets and subnets
│   └── 07-compute           ← Uses VNets, NSGs, subnets
│       └── 08-monitoring    ← Monitors VMs, storage, network
```

### 4. Follow the Lab Guide

```bash
# Open the comprehensive lab guide
open LAB-GUIDE.md
```

## Cleanup

> ⚠️ **Don't forget to clean up!** Azure resources incur costs even when idle.

See [CLEANUP.md](CLEANUP.md) for full cleanup instructions, or run:

```bash
# Destroy everything
./scripts/destroy-all.sh

# Destroy a specific module
./scripts/destroy-module.sh {module-name}
```
